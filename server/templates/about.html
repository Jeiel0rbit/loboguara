About the Project

Welcome to the Lobo Guará Project, a Cyber Threat Intelligence (CTI) platform designed to empower cybersecurity professionals with a comprehensive set of tools to enhance threat detection and response.

Our platform offers a range of key features:

  • SSL Certificate Search: Search for domains and subdomains using real-time SSL certificate data from trusted authorities.
  • SSL Certificate Discovery: Monitor newly issued certificates for specific keywords to detect potential threats early.
  • Tracking Link: Generate links that collect information from an attacker's device when accessed.
  • Domain Scan: Perform WHOIS lookups and subdomain enumeration for a given domain.
  • Web Path Scan: Identify URIs and web paths on specific URLs.
  • URL Scan: Capture screenshots and mirrors of web pages, useful for reporting malicious sites.
  • URL Monitoring: Monitor inactive URLs and trigger scans when they become active, ideal for tracking malicious sites.
  • Data Leak Alerts: Stay informed with real-time alerts on data leaks from hacker forums and websites.
  • URL+User+Password Search: Search leaked credentials across hacker forums to identify exposed user data.
  • Threat Intelligence Feeds: Centralized threat intelligence news from various sources to keep you updated.

We are committed to delivering value to the cybersecurity community, helping professionals improve their defenses against evolving cyber threats.
